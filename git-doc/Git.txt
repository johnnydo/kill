Git

You can get a Git project using 2 main approaches.
- The first takes an existing project or directory and imports it into Git.
- The second clones an existing Git repository from another server.

Initializing a Repository in an Existing Directory
$ git init
$ git add *.c
$ git add LICENSE
$ git commit -m "initial project version"

Cloning a Existing Repository
$ git clone https://github.com/ngocgiang2512/kill.git
